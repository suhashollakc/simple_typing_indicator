Simple Typing Indicator

API Reference

SimpleTypingIndicator

A StatefulWidget that displays three animated dots as a typing indicator.

Constructor

const SimpleTypingIndicator({
  Key? key,
  Color dotColor = Colors.grey,
  double dotSize = 8.0,
  double spacing = 4.0,
  Duration duration = const Duration(milliseconds: 1200),
  double speed = 1.0,
})

Parameters

dotColor (Color)
The color of each dot.
Default: Colors.grey
dotSize (double)
The width and height of each dot.
Default: 8.0
spacing (double)
The horizontal space between dots.
Default: 4.0
duration (Duration)
The base duration for one complete animation cycle.
Default: Duration(milliseconds: 1200)
speed (double)
A multiplier to adjust the animation speed. The effective duration is calculated as duration / speed.
Default: 1.0

Example Usage

// Basic usage
const SimpleTypingIndicator()

// Customized usage
const SimpleTypingIndicator(
  dotColor: Colors.blue,
  dotSize: 10.0,
  spacing: 5.0,
  duration: Duration(milliseconds: 1000),
  speed: 1.0,
)

Demo

Implementation Details

The widget uses a single AnimationController that repeats indefinitely. It defines three staggered animations using Interval to achieve the pulsing effect. The dot scaling animation transitions between 0.5 and 1.0 to simulate the "typing..." effect.

Notes

  • All parameters are optional and have sensible defaults.
  • The widget is optimized for performance and memory usage.
  • The animation automatically adjusts based on the speed parameter.
